#define WC_ASSERT (   e)    (e) ? (void)0 : WC_FAIL(WC_ASSERT_PREFIX #e)

Helper macro, proves expression satisfies true. In case expression evaluates to false a run-time error is issued.

Note
To use this macro you have to provide an implementation for the C function wc_Fail.

#define WC_ASSERT_PREFIX   "Assertion failed: "
