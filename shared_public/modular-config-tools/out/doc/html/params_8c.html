Function to evaluate early options (for example implementation see HelloWorld example config tool).

Prototype description
======================
Prototype to evaluate early options (options have to been checked before any further operations started).

Note
The function implementation has to be provided application specific.
As inner implementation ctparts_EvaluateEarlyOptions may be used.
Parameters
pstContextApplication specific context. If unused, NULL is allowed.
argcArgument count, forwarded from outer main function.
argvArgument vector, forwarded from outer main function.
pstOptionsOption structure pointer. Inside user specific option pointer ctutil_OptionsSpecific_t may be NULL.
Returns
CTUTIL_SUCCESS in case of success, an error code otherwise.

Function to allocate specific options struct and set the option defaults (for example implementation see HelloWorld example config tool).

Prototype description
======================
Prototype to set application specific default application options.

Note
The function implementation has to be provided application specific.
An eventually set pointer for a specific options structure have to be released by ctparts_ReleaseSpecificOptions.
Parameters
pstContextApplication specific context. If unused, NULL is allowed.
pstOptionsOption structure pointer to be equipped with defaults.
Returns
CTUTIL_SUCCESS in case of successfully set defaults, an error code otherwise.
Precondition
Application specific option pointer ctutil_OptionsSpecific_t is NULL.
Postcondition
Application specific option pointer ctutil_OptionsSpecific_t is NULL or a valid structure in case of success. Application specific option pointer ctutil_OptionsSpecific_t is NULL in case of an error.
