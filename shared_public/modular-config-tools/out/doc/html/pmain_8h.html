Early function for to prepare config tool main (for example implementation see HelloWorld example config tool).

Prototype description
======================
Prototype for config tool prepare main function.

Note
The function implementation has to be provided application specific.
This function has to call the specific config tool main function.
As inner implementation ctutil_CommonMain may be used. ctparts_ctMain.
Parameters
argcArgument count, directly forwarded from outer main function.
argvArgument vector, directly forwarded from outer main function.
Returns
Application exit code, CTUTIL_EXIT_SUCCESS on success, error code otherwise. The provided exit code should be positive and never a forward from a status code. The provided exit code should be between CTUTIL_EXIT_MIN and CTUTIL_EXIT_MAX. The provided exit code should honor the common exit code defines and avoid any conflict.
