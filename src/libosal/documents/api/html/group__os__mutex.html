Detailed Description

Mutexes are resources of the operating system to let threads share resources safely. Using a data structure with no locking, the system may run without problems for quite some time, but anytime the data structure will become inconsistent and the system behavior strangely.

Mutexes serve as a mutual exclusion mechanism between threads, and cannot be used to synchronize between threads and the interrupt handling subsystem.

To use the functionality of mutexes independent on an operating system the mutex specific routines, constants, types and function prototypes of the abstraction layer interface has to be use.

Routines:

  • os_mutex_create Create mutex
  • os_mutex_destroy Destroy mutex
  • os_mutex_lock Lock mutex
  • os_mutex_unlock Unlock mutex
