#define OS_HTON_32 (   x)    (uint32_t)OS_SWAP_32((uint32_t)x)

The OS_HTON_32 macro converts a 32bit value from the host byte order to the network byte order. The network byte order is always big-endian. The host byte order depends on the settings of the used machine and is little-endian on a little-endian machine and big-endian on a big-endian machine.

For example, the value 0x12345678 will be converted to 0x78563412 on a little-endian machine and will be not converted on a big-endian machine.

Parameters:
x32bit Specifies value to be swapped.
Returns:
The macro returns the value in network byte order.
