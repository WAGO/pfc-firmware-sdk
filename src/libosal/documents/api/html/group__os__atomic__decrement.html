int_t os_atomic_decrement ( int_t *  target)

The routines decrements (decreases by one) the value of the specified variable as an atomic operation. The variable pointed to by the address target must be aligned on a 32-bit boundary.

Parameters:
targetPointer to the variable to decrement.
Returns:
The return value is the resulting decremented value.
