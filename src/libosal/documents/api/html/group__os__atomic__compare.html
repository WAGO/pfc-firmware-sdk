Function Documentation

int_t os_atomic_compare ( int_t *  target,
int_t  compare,
int_t  exchange 
)

The method performs an atomic comparison of the specified values and exchanges the values based on the outcome of the comparison. If the target value is equal to the compare value, the exchange value is stored in the address specified by target. Otherwise, no operation is performed. The variables for os_atomic_compare must be aligned on a 32-bit boundary.

Parameters:
targetSpecifies the address of the destination value.
compareSpecifies the value to compare to target.
exchangeSpecifies the exchange value.
Returns:
The return value is the initial value of the destination target.
