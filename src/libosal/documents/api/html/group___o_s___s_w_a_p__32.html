Define Documentation

#define OS_SWAP_32 (   x)
Value:
((((x)<<0x00000018)&0xFF000000)|\
                                   (((x)<<0x00000008)&0x00FF0000)|\
                                   (((x)>>0x00000008)&0x0000FF00)|\
                                   (((x)>>0x00000018)&0x000000FF))

Calling OS_SWAP_32 swaps the highest byte of the 32bit value with the lowest byte and the second byte with the third byte.

For example, the value 0x12345678 will be converted to 0x78563412.

Parameters:
x32bit Specifies value to be swapped.
Returns:
The macro returns the swapped value.
