Modules

 os_memory_alloc
 os_memory_dealloc

Detailed Description

The provided memory functionality of the operating system abstraction layer allows using data memory dynamically. Use these routines of the abstraction layer to allocate and release memory independent on the operating system.

Routines:

  • os_memory_alloc Allocate memory
  • os_memory_dealloc Deallocate memory
