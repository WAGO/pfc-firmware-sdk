OS_SWAP_161

Defines

#define OS_SWAP_161(x)   (((x)<<0x0008)&0xFF00)

Define Documentation

#define OS_SWAP_161 (   x)    (((x)<<0x0008)&0xFF00)

Calling OS_SWAP_161 changes the position of the lower byte of the 16-bit parameter value to the higher byte position. The purpose of this macro is mainly the use by the OS_HTON_16 and OS_NTOH_16 macros for endian conversion. Example: value 0x1234 is made to 0x3400.

Parameters:
x16bit value to be swapped
Returns:
swapped value
