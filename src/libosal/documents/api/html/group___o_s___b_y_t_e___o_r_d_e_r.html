Byte Order Macros

Modules

 OS_HTON_16
 OS_HTON_32
 OS_NTOH_16
 OS_NTOH_32

Detailed Description

The macros converts values greater than one byte from from host to network byte order or from network to host byte order. The conversion depends on the machine (little-endian or big-endian).

The macros are usually required by parsing or building PDUs.

Macros:

  • OS_HTON_16 - convert from host to network byte order (16bit value)
  • OS_HTON_32 - convert from host to network byte order (32bit value)
  • OS_NTOH_16 - convert from network to host byte order (16bit value)
  • OS_NTOH_32 - convert from network to host byte order (32bit value)
