typedef uint32_t(* os_thread_routine_t)(void *user)

The type specifies the function prototype of the thread execution routine. The routine is an application-defined routine, which serves as the starting address for a thread. The address has to be specified by calling the os_thread_create routine.

Parameters:
pParamspecifies the thread data passed to the function using the user parameter of the os_thread_create routine
Returns:
The thread execution routine should return a value that indicates its success or failure.
