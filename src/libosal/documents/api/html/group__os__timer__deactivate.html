void os_timer_deactivate ( handle_t  ref)

A call to the os_timer_deactivate deactivates a timer. The timer stops timing the interval immediately. After the routine returns a time-out event will not be notified anymore. The timer can't be reactivated after calling this function.

Parameters:
refSpecifies the timer related reference of the timer.
