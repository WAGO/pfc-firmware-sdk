int_t os_atomic_increment ( int_t *  target)

The routines increments (increases by one) the value of the specified variable as an atomic operation. The variable pointed to by the address target must be aligned on a 32-bit boundary.

Parameters:
targetPointer to the variable to increment.
Returns:
The return value is the resulting incremented value.
