handle_t os_timer_restart_onetime ( handle_t  ref,
uint_t  timeout 
)

Calling os_timer_restart_onetime restarts a timer for a single shot after which it is stopped again. The timer starts timing the interval as soon as it is activated. For each event the elapsed timer notifies the user by using the registered callback routine os_timer_activate_onetime.

A timer which is not longer required can be destroyed by using the os_timer_deactivate routine.

Parameters:
refSpecifies the timer related reference of the timer.
timeoutSpecifies the time-out value, in milliseconds. If this value is not in the range of the minimum and maximum event delays supported by the timer, the function returns an error.
Returns:
A reference to the timer (parameter ref) indicates success. NULL indicates failure.
