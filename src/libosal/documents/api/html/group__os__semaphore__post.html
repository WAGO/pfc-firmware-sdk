void os_semaphore_post ( handle_t  semaphore)

A thread uses the os_semaphore_post routine when an event has occured. The operation increments the counter and releases a thread waiting on the semaphore. The waked up thread continues running and the os_semaphore_wait routine decrements the counter again before returning.

Parameters:
semaphoreSpecifies the reference to the semaphore returned by a successful call to os_semaphore_create.
