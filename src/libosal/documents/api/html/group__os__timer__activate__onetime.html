handle_t os_timer_activate_onetime ( handle_t  user,
uint_t  timeout,
os_timeout_routine_t  routine 
)

Calling os_timer_activate_onetime activates a timer. This timer starts timing the interval as soon as it is activated. The timer is set to expire only once. For each event the elapsed timer notifies the user by using the registered callback routine. After expiring the timer is stopped and can be restarted by calling os_timer_restart_onetime or os_timer_restart_periodical.

A timer which is not longer required can be destroyed by using the os_timer_deactivate routine.

Parameters:
userSpecifies the timer related reference of the user.
timeoutSpecifies the time-out value, in milliseconds. If this value is not in the range of the minimum and maximum event delays supported by the timer, the function returns an error.
routinePointer to a callback routine which is called once on expiration of a single event or periodically on expiration of periodic events. The parameter is interpreted as a function pointer. The prototype is specified at os_timeout_routine_t.
Returns:
A reference to the timer indicates success. NULL indicates failure.
