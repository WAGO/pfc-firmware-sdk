Detailed Description

Routines:

  • OS_SWAP_161 - swap low byte to high byte (16bit value)
  • OS_SWAP_162 - swap high byte to low byte (16bit value)
  • OS_SWAP_16 - swap bytes (16bit value)
  • OS_SWAP_321 - swap lowest (4) byte to highest (1) byte (32bit value)
  • OS_SWAP_322 - swap byte 3 to byte 2 (32bit value)
  • OS_SWAP_323 - swap byte 2 to byte 3 (32bit value)
  • OS_SWAP_324 - swap byte 1 to byte 4 (32bit value)
  • OS_SWAP_32 - swap bytes (32bit value)
  • OS_HTON_16 - convert to network order (16bit value)
  • OS_HTON_32 - convert to network order (32bit value)
  • OS_NTOH_16 - convert to host order (16bit value)
  • OS_NTOH_32 - convert to host order (32bit value)
