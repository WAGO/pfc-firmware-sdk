OS_SWAP_322

Defines

#define OS_SWAP_322(x)   (((x)<<0x00000008)&0x00FF0000)

Define Documentation

#define OS_SWAP_322 (   x)    (((x)<<0x00000008)&0x00FF0000)

Calling OS_SWAP_322 changes the position of the third byte of the 32-bit parameter value to the second byte position. The purpose of this macro is mainly the use by the OS_HTON_32 and OS_NTOH_32 macros for endian conversion. Example: value 0x12345678 is made to 0x00560000

Parameters:
x32bit value to be swapped
Returns:
swapped value
