Detailed Description

Threads are resources of an operating system to execute code. Additional threads are created to wait for several events to occur so another threads are not blocked.

To use the functionality of threads independent on an operating system the thread specific routines, constants, types and function prototypes of the abstraction layer interface has to be use.

Routines:

  • os_thread_create Create thread
  • os_thread_destroy Destroy thread
  • os_thread_exit Terminate thread for itself

Prototypes:

  • os_thread_routine_t Thread execution routine
