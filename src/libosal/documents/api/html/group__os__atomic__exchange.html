Function Documentation

int_t os_atomic_exchange ( int_t *  target,
int_t  value 
)

The routine sets an integer variable to a given value as an atomic operation. The os_atomic_exchange routine is atomic with respect to calls to other atomic routines. The variables for os_atomic_exchange must be aligned on a 32-bit boundary.

Parameters:
targetPointer to the value to exchange. The function sets this value to value and returns its prior value.
valueSpecifies the value for the value pointed to by target.
Returns:
The function returns the value of the variable at target when the call occurred.
