void os_time_get_time ( uint32_t *  time_s,
uint32_t *  time_us 
)

Calling os_time_get_time returns the value of a continuously running time counter starting at 0 on system startup.

The maximum counter value of the continuous counting tick clock is 4,294,967,294 (e.g. 2^32-2). The clock automatically resets to 0 on the tick after the maximum value is reached. As the parameter values are directly calculated from the tick value of the system clock, values time_s and time_us are both reseted 0 at the same time. The user has to consider this when calculating time differences from the clock values.

Parameters:
time_sPoints to an address to store the seconds of the time value.
time_usPoints to an address to store the microseconds of the time value.
