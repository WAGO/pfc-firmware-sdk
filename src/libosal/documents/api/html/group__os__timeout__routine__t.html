typedef void(* os_timeout_routine_t)(handle_t user)

The type specifies the function prototype of the time-out routine. The routine is an user-defined callback routine, which is called once on expiration of a single event or periodically on expiration of periodic events.

Parameters:
userSpecifies a reference to user specific data passed to the function using the user parameter of os_timer_activate_periodical.
