void os_memory_dealloc ( void *  mem)

The os_memory_dealloc routine deallocates a memory block specified by pMem that was previously allocated by a call to os_memory_alloc. The number of freed bytes is equivalent to the number of bytes requested when the block was allocated. If pMem is NULL, the pointer is ignored and the routine immediately returns.

Parameters:
memPreviously allocated memory block to be deallocated.
